## 🍀 教程大纲

整个教程分两个部分进行讲解：第一部分，主要介绍如何在本地搭建一个联盟链环境，以及在链运维过程常见的操作，比如：添加联盟节点，添加共识节点等；第二部分，主要介绍如何基于联盟链开发一款DApp应用，包括SDK的集成、测试用例、后端服务以及前端展现等等。
